Reference Checker for Facebook Videos

Verify claims made in Facebook videos with real-time Reference-checking.

What Does It Do?

Our Chrome extension analyzes Facebook videos in real-time, checking claims made in videos against reliable sources. It helps you identify misinformation and find trustworthy references.

  • Real-Time Reference-Checking: Automatically detects claims in Facebook videos.
  • Reliable References: Provides supporting or debunking references from trusted sources.
  • Improve Online Literacy: Get insights and verify information quickly.

Why Use It?

In an era of widespread misinformation, our tool ensures you can verify the information in Facebook videos as you watch, helping to stop the spread of false claims.

How to Install the Extension

  1. Download the Extension
    • Go to the GitHub repository and download the extension's source code.
    • Click the green Code button, then select Download ZIP.
  2. Extract the ZIP File
    • Find the ZIP file in your downloads folder and extract its contents.
  3. Add the Extension to Chrome(official chrome documentation)
    • Open Chrome and go to Extensions (type chrome://extensions/ in your address bar).
    • Enable Developer mode by toggling the switch at the top.
    • Click on Load unpacked and select the extracted folder.
  4. Start Reference-Checking!
    • Once loaded, the extension will appear in your Chrome toolbar.
    • Go to Facebook, watch videos, and get real-time Reference-checking.

FAQ

Is this extension free to use?

Yes! The extension is completely free.

How does the Reference-checking work?

The extension uses speech-to-text technology and checks claims against a database of trusted sources.

Can it be used on other platforms?

Currently, it’s optimized for Facebook, but we’re working on expanding support to other platforms.

Is my data safe?

Yes. Your data is not stored or shared. The extension only analyzes video content locally in your browser.
